# Decision Matrices -- Strange's Reference

## Weighted Decision Matrix
When comparing multiple options across several criteria:
1. List criteria (columns) and options (rows)
2. Weight each criterion by importance (1-5)
3. Score each option per criterion (1-5)
4. Multiply score x weight, sum across
5. Highest total = recommended option, but check gut feel too

## Eisenhower Matrix (Priority)
- Urgent + Important = Do now
- Not Urgent + Important = Schedule (these get neglected most)
- Urgent + Not Important = Delegate
- Not Urgent + Not Important = Eliminate

## Risk Assessment Matrix
- Likelihood (1-5) x Impact (1-5) = Risk Score
- Score > 15: Must address before proceeding
- Score 8-15: Mitigate with a contingency plan
- Score < 8: Accept and monitor
